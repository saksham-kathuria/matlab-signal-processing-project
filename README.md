# MATLAB Signal Processing and Noise Filtering

This is a simple illustration of MATLAB Signal Processing.

## Highlights:
- Signal creation (Sine Wave)
- Addition of noise (Gaussian)
- Filtering of noise (Moving Average)
- Graphical representation

## Software Utilized:
- MATLAB

## Results:
Presentation of original, noise-infected, and noise-filtered signals.# matlab-signal-processing-project
In this project, we will explore basic signal processing by implementing our processes through MATLAB. In the first step, we create a signal in a sine wave shape but deliberately introduce Gaussian noise in it to represent disturbances in the real world. Next, we implement a moving average filter to help remove noise from the signal.
